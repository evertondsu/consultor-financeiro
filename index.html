Conectar ao GitHub Gist
Seus dados serão salvos em um Gist privado na sua conta GitHub.
Isso permite acessar o painel de qualquer dispositivo.
Como obter o token?
1. Acesse github.com → Settings → Developer settings
2. Clique em Personal access tokens → Tokens (classic)
3. Clique em Generate new token
4. Marque apenas a permissão gist
5. Copie e cole o token acima
